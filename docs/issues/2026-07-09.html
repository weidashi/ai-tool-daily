web-access — Codex 的「联网之眼」
eze-is · 全平台联网搜索 + 浏览器自动化
⭐ 8.1K联网搜索浏览器自动化Codex 兼容
是什么：一个让 AI Agent 获得完整联网能力的 Skill。不只是简单的 WebSearch / WebFetch，而是通过 CDP 协议直连你的 Chrome/Edge 浏览器，天然携带登录态——能搜网页、能读内容、能操作交互页面，甚至能做并行多目标调研。
最适合：需要搜集行业资料、做竞品调研、批量比对信息的场景。知识工作者的「研究助理」，再也不用手动翻几十个网页。
📦 安装
npx skills add eze-is/web-access
自动检测 Codex 环境安装到正确位置。需要 Node.js 22+，CDP 模式需 Chrome/Edge 开启远程调试
🎯 今天就能试：在 Codex 里说「用 web-access 帮我同时调研钉钉、飞书、企微三家的 AI 功能最新动态，给我一份对比摘要」——它会自动并行打开三个官网，提取关键信息，汇总为一张对比表。
📌 实操示例
输入：帮我搜索「2026 下半年 AI 写作工具趋势」，看看最近有哪些新产品或功能更新，整理一份 500 字简报
成果：一份结构化的 AI 写作工具趋势简报，包含 3 个重点产品的功能对比、用户反馈归纳和我的判断。每条信息附带来源链接，可直接用于选题调研。

Daily-paper — 论文追踪不费脑，每天自动发到邮箱
hhhzauuu · 每日论文检索 + 自动中文摘要 + 邮件推送
⭐ 新发布论文追踪Codex 自动化邮件推送
是什么：一个面向科研阅读的 Codex 自动化项目。配置你的研究方向后，Codex 每天自动检索 arXiv、OpenReview、ACL Anthology 等学术来源的最新论文，下载 PDF、提取主图、写中文 digest、更新 Excel 汇总表，最后把整理好的摘要发到邮箱。
最适合：需要持续追踪某个领域前沿的人——学术研究者、行业分析师、咨询顾问、产品经理。不用每天手动刷 arXiv，早上一杯咖啡的时间，邮件里已经有 3-5 篇真正值得读的论文摘要。
📦 安装
git clone https://github.com/hhhzauuu/Daily-paper.git && cd Daily-paper && cp -Recurse -Force skills/daily-paper-digest "$env:USERPROFILE\.codex\skills\daily-paper-digest"
需要 Python 3.10+、SMTP 邮箱、支持 GPT web search 的 Codex 模型。安装后在 Codex 说「使用 daily-paper-digest skill，帮我检查项目配置是否可用」
🎯 今天就能试：配置研究方向为「企业战略咨询方法论」，每天早 8 点自动收到 3-5 篇最新相关论文的中文摘要，含 PDF 链接 + 主图 + 你的理解 + 局限分析。Excel 累积一段时间后就是你的专属研究文献库。
📌 实操示例
输入：配置 research_topic 为「数字化转型对国有企业组织效能的影响」，keywords 含「国企改革」「组织变革」「数字化成熟度」
成果：每天早上 8 点邮件推来 3-5 篇最新论文，每篇含：标题+作者+来源+PDF链接+中文摘要+结论翻译+My Understanding+局限分析。周末翻一翻 Excel 汇总表，一周的学术动向一目了然。

imagegen — 写文章时配图不用愁，让 AI 帮你画
Codex 内置 · 文本生成图片，零安装零配置
⭐ Codex 内置AI 生图零安装
是什么：Codex 自带的图片生成能力，不需要额外安装任何 Skill。在对话中描述想要的画面——可以是公众号封面图、文章配图、概念图解、数据示意图——Codex 直接生成图片。支持照片级写实、插画风格、信息图解等多种风格。
最适合：写文章需要配图但不想学设计工具；做 PPT 缺视觉素材；需要快速画一张概念图来解释复杂想法。不用打开 Canva、不用找图库、不用调 PS。
📦 触发方式
直接在 Codex 对话中说「帮我生成一张图：XXX」
无需安装，Codex 内置。描述越具体越好：风格、色调、构图、元素等。
🎯 今天就能试：对 Codex 说「帮我生成一张公众号封面图：主题是"信息获取力"，风格简约商务，深蓝+金色配色，包含一本书和放大镜的意象，2.35:1 宽幅」——30 秒后你就有了一张独一无二的封面，不用翻图库。
📌 实操示例
输入：帮我生成 3 张文内配图，主题「知识工作者的一天」：第一张是清晨喝咖啡读邮件，第二张是中午用 AI 工具分析数据，第三张是傍晚整理知识笔记。风格：扁平化插画，暖色调，16:9
成果：3 张风格统一的扁平化插图，可直接插入文章。文章从纯文字变成图文并茂，阅读体验翻倍。不用找图库、不担心版权。

article-writer — 从搜索到成文：11 步系统化写作流
H1N2/wordflowlab · 13 平台兼容 · 教练/快速/混合三种写作模式
⭐ 热门系统写作Codex CLI11 步流水线AI 味 <30%
是什么：一套专为内容创作者设计的系统化写作工具。通过斜杠命令（/specify → /topic → /research → /write → /review → /publish）把写作拆成 11 个可控步骤。三种模式可选——教练模式（AI 只提问引导，0% AI 生成）、快速模式（AI 生成初稿）、混合模式（AI 写框架你来填）。支持公众号、视频脚本、通用内容三种工作区。
最适合：需要写深度文章但不想「从头到尾全靠 AI 写然后全是 AI 味」的人。教练模式下 AI 检测率可低至 15% 以下，写出来的文章读起来完全不像机器写的。
📦 安装
npm install -g article-writer-cn && content init my-article --workspace wechat --ai codex
安装后初始化项目，在 Codex 中使用 /content-write 等斜杠命令触发。Codex CLI 形式为 /content-命令名。
🎯 今天就能试：在 Codex 里用「/content-topic」讨论你要写的选题，AI 会给 3-4 个方向供你选；确定方向后用「/content-research」自动搜索资料，最后用「/content-write」选择教练模式——AI 不帮你写一个字，只提问引导你理清思路，最终文章 AI 检测率 <15%。
📌 实操示例
输入：/content-write 教练模式，主题「AI 时代我们还需要记笔记吗」。AI 会问：你的核心观点是什么？你自己用什么笔记工具？有没有具体的一个例子可以展开？
成果：一篇 2500 字的公众号文章，用了你自己的经历和真实例子，没有一句套话。发给朋友看，没人能看出 AI 参与过——因为 AI 只负责问问题，内容全是你自己写的。

⚡ Skill 管理技巧：按任务场景分组，不全部挂载
一个常见误区是「越多 Skill 越好」。实际上每个 Skill 的 prompt 描述在每次 Codex 调用时都会被加载，10 个 Skill 同时挂着还没开始干活 context 就吃满了。建议策略：按任务分场景——写作场景只挂写作相关 Skill，研究场景只挂搜索和分析 Skill。跑完一个任务再换一批。

NotebookLM 每日进步
🧪 NotebookLM 6 月大更新：研究能力全面进化
Google 在 6 月 8 日推送了 NotebookLM 史上最大一次更新。核心变化：底层模型升级为 Gemini 3.5（理解力质的飞跃）、每个 notebook 配备云端代码执行环境（终于能做数据分析了）、新增「推理可见」模式（chat 里会展示它的思考步骤，不再黑箱）、Agent 自主发现信源（给它一个研究方向，它自己去找资料建库）。新导出格式也补上了——图表可导出为 PNG/SVG，文档支持 PDF/Word/Markdown/Excel/PPT 等。
📊 新玩法：用「Agent 自主建库」做竞品研究
无需手动上传文件了。给 NotebookLM 一个研究方向，比如「中国企业战略咨询行业竞争格局」，它会自动全网搜索相关信源，建立专属资料库。之后基于这个库做深度问答、生成简报、出思维导图——全程不需要你手动找资料。这个功能目前仅对 Google AI Ultra 和 Workspace 企业用户开放，但免费版迟早会跟上。
🎧 Audio Overview 暂时没有升级——但它有了新出口
NotebookLM 生成的双人对话播客（Audio Overview）在 6 月更新中没有任何改动——不能编辑脚本、不能自定义主播、不能导出到播客平台。但已经有工具补齐了这个缺口：Jellypod 等第三方服务可以接收 NotebookLM 的 Audio Overview 文件，转录并分离两个 speaker，让你编辑脚本后用自己的声音重新生成，再推送到 Spotify、Apple Podcasts。你的 NotebookLM 播客终于能变成真正的「节目」了。

WorkBuddy 实用技巧
⚡ 多模型混用：不同任务配不同的「大脑」
WorkBuddy 内置混元、DeepSeek、Kimi 等多个模型，在对话框右上角一键切换，无需重启。搭配策略：混元——擅长办公文档、PPT、中文创作，写周报和汇报材料用它；DeepSeek——擅长数据分析、复杂计算，处理 Excel 报表用它；Kimi——擅长长文档解读、PDF 分析，读万字报告用它。一句话总结：不要只用默认模型，把不同模型当不同特长的同事来调配。
🧩 Skill 市场挖宝：别只盯着官方技能
WorkBuddy 自带 30+ 技能，但真正好用的很多是社区贡献的。左侧菜单进入「技能市场」，可以搜索安装。推荐四件套：办公套件（Word/Excel/PPT/PDF）、联网搜索、浏览器操控、邮件连接器。全部免费安装，即装即用。安装方式：在对话中直接说「帮我安装 XX 技能」，或去市场搜索。
