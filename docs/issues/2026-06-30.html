公众号写作 Skills
①
wechat-article-write — 公众号文章创作节点
classfang · 选题→大纲→正文→视觉风格→配图 prompt，一条路走完
⭐ 新发布公众号写作全流程Codex 原生
是什么：classfang/weixin-mp-skills 合集里的文章创作 Skill。给它一个选题或大纲，它会依次完成：理解选题方向 → 生成标题和大纲 → 撰写正文 → 匹配视觉风格 → 输出封面图和文内配图 prompt。你只需要说「帮我写一篇关于 XX 的公众号文章」，剩下的全交给它。
最适合：有选题但不知道怎么展开的公众号作者、从大纲到成文的一站式创作、需要视觉风格建议的内容创作者。
📦 安装
git clone https://github.com/classfang/weixin-mp-skills.git && cd weixin-mp-skills && ./scripts/update-skills.sh
💡 一键脚本会把 6 个 Skill 全部同步到 ~/.codex/skills/。也可单独 cp wechat-article-write ~/.codex/skills/。安装后在 Codex 里说「使用 $wechat-article-write 写一篇 XX 主题的文章」。
🎯 今天就能试：把你的选题发给 Codex → 「使用 $wechat-article-write，帮我写一篇关于「AI 工具如何改变公众号写作」的 2000 字深度文章，目标读者是 25-35 岁的内容创作者」→ 5 分钟拿到标题+大纲+正文+配图建议。
②
dbskill — 选题诊断器（公众号视角）
dontbesilent · 诊断选题是否成立、优化标题和 hook
⭐ 高收藏选题诊断标题优化
是什么：专为中文内容创作者设计的选题诊断工具。喂给它 3 个待选选题，它会告诉你哪个「结构性最优」——受众明确、痛点清晰、有信息差。还会推荐 3 版标题和 2 个 hook 方向。
公众号用法：写公众号文章之前，先用 dbskill 确诊选题值不值得写。避免花 2 小时写了一篇没人看的文章。
📦 安装
npx -y skills@latest add dontbesilent/dbskill --skill dbskill --agent codex --global
🎯 今天就能试：列出 3 个选题 → 「用 dbskill 诊断这 3 个公众号选题，目标读者是 XX 群体」→ 4 分钟拿到判断+标题+hook。

NotebookLM & Notion 每日进步
🧪 NotebookLM — 用「对比」问题挖出矛盾观点
上传多篇来源后，问「A 和 B 在 XX 观点上有什么矛盾？」比问「这些文章讲了什么」得到的信息密度高 3 倍。因为 NotebookLM 会逐句检索冲突点，而不是泛泛归纳。写公众号选题时，把 3 篇不同立场文章喂进去，问矛盾点——选题角度自动浮现。
🧪 NotebookLM — Studio 创作套件：写+排版+导出一条路
6 月新增的 Studio 模式把写作、排版和多格式导出串成一条路。写完笔记后可以直接导出 PPT、PDF、Excel 或图片——不用切换工具。对公众号作者来说，这意味着在 NotebookLM 里做完研究，导出 PDF 就能发给同事审稿。
🧪 NotebookLM — 给 AI 播客加「自定义指令」
生成播客前输入自定义指令可以精准控制内容——比如「只讲第三章的实验数据对比」或「用小学生能听懂的方式解释」。默认播客容易把简单的事讲绕，加了指令后质量提升明显。
📋 Notion — Agent 定时自动执行任务
Notion 3.0 的 Custom Agent 现在支持定时触发：设置「每天早上 9 点自动整理昨天的会议纪要到项目页面」，或「每周一自动从数据库提取数据生成周报」。不需要你手动跑，Agent 在后台按时干活。对公众号运营来说，可以设置定时整理选题灵感库。
