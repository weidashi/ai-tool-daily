学术研究五件套
①
deep-paper-note — 英文论文深度笔记
社区贡献 · 从摘要到实验设计，逐层拆解
⭐ 4K+论文学术
是什么：把一篇 PDF 论文转化为结构化笔记——研究问题→方法→实验设计→关键发现→局限→启发。GPT-5.6 对学术语境的理解和术语一致性保持比 GPT-5 强一大截，不会出现"前言说的方法是 A，结论里变成了 B"的低级错误。
最适合：每周需要精读 2-5 篇论文的研究生、分析师、技术从业者。

npx -y skills@latest add githubfans518/deep-paper-note --skill deep-paper-note --agent codex --global

社区贡献 · 在 Codex 里直接操控 NotebookLM
⭐ 3K+工具

最适合：同时用 Codex 和 NotebookLM 做研究的人——信息不用搬来搬去。

daily-paper — 每日论文追踪
社区贡献 · 设定关键词，每天推送新论文
⭐ 2K+追踪
是什么：设定研究领域和关键词，每天自动抓取 ArXiv、Semantic Scholar 上新发表的论文，并生成一句话摘要。GPT-5.6 的摘要能力让每篇论文的一句话总结更精准。
最适合：不想手动刷 ArXiv 但在意领域最新进展的研究者。
📦 安装
npx -y skills@latest add githubfans518/daily-paper --skill daily-paper --agent codex --global
④
web-access — Codex 联网搜索
内置能力 · 搜索最新论文和行业数据
⭐ 内置搜索
是什么：GPT-5.6 的联网搜索能力在内置 Skill 中可用——让 Codex 实时搜索最新论文、行业报告、开源数据集。搭配 deep-paper-note 形成"搜→读→记"闭环。
最适合：研究起步阶段——"帮我找 2026 年关于 LLM 推理能力的最新 5 篇论文并做笔记"。

最适合：用 Obsidian 做知识管理的研究者——论文笔记不再散落在各种文件夹里。

GPT-5.6 学术能力实测
📐 长论文理解：40 页 PDF 不再"翻到后面忘了前面"
GPT-5 的上下文窗口虽然大，但长文档中信息丢失明显。GPT-5.6 的 persisted reasoning 让长论文分析保持全程一致——前面提到的方法在后面的分析中仍然被正确引用。
🔗 跨论文关联：发现不同论文之间的隐性对话
把 3 篇同领域的论文同时输入，GPT-5.6 能找出它们之间的观点分歧、方法差异和引用的共同源头——这在 GPT-5 上需要手动对比。
