Codex 日常效率 Skills
①
content-research-writer — 深度内容研究写作
ComposioHQ · 从搜索到成文一站式搞定
⭐ 12K+写作研究
是什么：这个 Skill 被 7 类创作者一致点名推荐——文案、编辑、自媒体、课程设计、广告策划、产品运营、知识博主，所有人的标配。它会自动搜索资料、搭建文章框架、逐段写初稿。
最适合：写公众号深度文章、行业分析报告、课程大纲——任何需要"先研究再输出"的内容任务。
📦 安装
npx -y skills@latest add ComposioHQ/awesome-codex-skills --skill content-research-writer --agent codex --global
🎯 今天就能试：让它帮你研究一个你一直想写但拖延的选题，比如"AI 对知识工作者的影响"，5 分钟拿到一份带提纲的研究初稿。
📌 实操示例
输入："帮我研究2026年AI对知识工作者的影响趋势，目标读者是25-35岁职场人，写一篇2000字的分析文章，分4个板块：效率提升、角色变化、技能要求、未来展望"
成果：自动搜索相关数据 → 生成4段提纲 → 逐段展开初稿。你只需要审核和微调，不用从零开始搜资料。
②
humanizer-zh — 中文去AI味神器
op7418 · 让AI写的文章读起来像人写的
⭐ 9.5K中文优化写作润色
是什么：专门针对中文 AI 写作模式优化的去味工具。它会自动识别并删除"首先…其次…最后"三段论、"值得注意的是""总体而言"等AI腔，替换为更自然的表达。
最适合：公众号文章、小红书文案、报告总结——任何你不想让人一眼看出是AI写的内容。
📦 安装
npx -y skills@latest add op7418/Humanizer-zh --skill humanizer-zh --agent codex --global
🎯 今天就能试：把你最近一篇AI辅助写的文章丢给它，5 步处理后对比原文——你会惊讶于"AI味"有多重。
③
official-document-drafting — 公文写作规范化
zhaohui-yang · 22 种公文模板，一键生成标准格式
⭐ 1.5K公文Word导出
是什么：按国家《党政机关公文格式》(GB/T 9704-2012) 规范起草报告、通知、请示、纪要等 22 种公文，可直接导出 Word。支持 Skill 模式、网页版 AI 模式、内网离线三种用法。
最适合：国企/事业单位/政府部门的日常公文起草，格式、用语、结构一键规范。
📦 安装
npx -y skills@latest add zhaohui-yang/official-document-drafting --agent codex --global
🎯 今天就能试：试试让它起草一份"关于推进数字化转型工作进展的报告"，格式规范程度让你不再被领导挑格式毛病。
④
remove-ai-flavor — 极简去AI味
B1lli · 专注清理AI写作痕迹，不做多余润色
⭐ 1K+精简体极简
是什么：和 humanizer-zh 不同，这个 Skill 的哲学是"只管去味，不改原意"。它只清理AI写作的结构痕迹——二分对照壳（"不是…而是"）、机械顺序壳（"首先…其次"）、本质拔高壳（"真正重要的是"）、假互动结尾（"你觉得呢?"）等。
最适合：你已经对内容满意，只需要"洗掉AI味"的场景。不改语气、不改风格、不改观点。
📦 安装
git clone https://github.com/B1lli/remove-ai-flavor-writing-skill.git ~/.codex/skills/remove-ai-flavor
🎯 今天就能试：把你ChatGPT生成的内容扔进去，它只删"壳"，不碰"肉"——处理后的文字可以放心发。
⑤
canvas-design — 排版设计标准化
ComposioHQ · 卡片、海报、封面、长图全搞定
⭐ 5K+设计排版
是什么：让 AI 帮你做排版布局——信息层级、画布结构、视觉动线。不管你是做小红书卡片、公众号封面、还是PPT页面，它能给出专业级的layout建议和HTML生成。
最适合：非设计师需要快速出图的场景。你不用懂设计原理，描述想要的感觉，它给你可用的layout。
📦 安装
npx -y skills@latest add ComposioHQ/awesome-codex-skills --skill canvas-design --agent codex --global
🎯 今天就能试："帮我设计一张公众号封面图，主题是'2026下半年工作规划'，简约商务蓝白配色，900x383"

NotebookLM 每日进步
🧪 用 Video Overview 5 分钟本地化一份PPT
把别人的 .pptx 上传到 NotebookLM，用你想要的语言生成 Video Overview，5分钟内就能得到一份带解说和幻灯片的本地化版本。不需要动 PowerPoint，不需要录旁白。适合快速学习别人的演讲内容。
🔍 让NotebookLM帮你发现"来源之间的矛盾"
一个被低估的用法：在 NotebookLM 中同时上传多份研究资料，然后问"这些来源在XXX问题上有什么矛盾或不同观点？" AI 会逐篇对比，找出你手动阅读可能忽略的冲突点。
📝 善用"Save Note"做模板化提问
对于重复性的研究任务（如每次项目都要问"客户核心诉求""价格敏感度""决策链"），把这些问题保存为 Saved Note。新建项目笔记本时直接复用，效率翻倍。

WorkBuddy 实用技巧
⚡ 定时任务 + 微信推送：让AI替你值班
WorkBuddy 的自动化面板支持 Cron 表达式的定时任务，配置后到点自动执行。比如早上8:30自动抓行业新闻推送到企业微信、中午12:00汇总待办事项、下午5:30生成日报草稿。设一次，长期托管，再也不会漏。
🔄 不满意不用重来，直接"迭代指令"
很多新手一不满意就重新输入指令，浪费积分。正确做法是基于上一轮结果增量修改："格式调整为报告风格""语言更简洁""补充一组对比表格""按小红书风格改写"。WorkBuddy 支持上下文继承，省积分又省时间。
