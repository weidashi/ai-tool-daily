写了半天好内容，发出去没人看——问题往往不出在文字上，而是视觉表达跟不上内容质量。今天推荐的4个Skill覆盖了从"一句话出PPT"到"选题雷达自动抓热点"，让非设计师也能做出过目不忘的视觉内容。全部 Codex 原生，一行命令安装。

Codex 视觉设计 Skills
①
huashu-design（华书设计）
alchaincyf · 一句话出PPT/原型/动画/信息图，20种设计哲学驱动
⭐ 13.5KMIT开源跨Agent通用
是什么：把设计能力装进Codex。不像传统"AI做PPT"只套模板——它内置20种设计哲学（瑞士现代、纸墨风格、暗色植物、赛博霓虹等），先给你3个风格方向让你挑，再按你的选择生成成品。能产出6类交付物：交互原型、HTML幻灯片+可编辑PPTX、时间轴动画MP4、信息图、设计方向顾问、5维度专家评审。
最适合：不会设计但需要高颜值交付物的人——产品发布、客户提案、内部汇报、公众号封面大图。
📦 安装
npx skills add alchaincyf/huashu-design
跨Agent通用：Codex、Claude Code、Cursor、OpenClaw、Hermes 都能装，MIT开源免费商用
🎯 今天就能试：打开Codex，说「用huashu-design把我这篇文章做成一套演讲稿PPT，推荐3个风格方向让我选」，AI先生成3个预览页，你挑一个，它再出完整15页deck+可编辑PPTX。
📌 实操示例
输入：「把这篇关于AI赋能国企战略转型的3000字分析文章做成演讲幻灯片，受众是国企中高层，要专业不花哨，推荐3个风格方向让我选」
成果：3套风格预览（瑞士现代/纸墨/暗色稳重）→ 选纸墨风格 → 12页HTML幻灯片+可编辑PPTX+演讲者备注。全程15分钟，以前手动排版至少2小时。

最适合：需要把复杂概念讲清楚的场景——架构说明、数据对比、项目复盘、研究简报。

NotebookLM（现 Gemini Notebook）每日进步
🔄 NotebookLM 正式更名 Gemini Notebook + 解锁代码执行（7月19日）
Google宣布NotebookLM更名为Gemini Notebook，同时解锁重磅能力——每个笔记本都配置了一台安全的云端计算机，能在后台写代码、跑代码来分析你的资料。以前它只能"读"和"总结"，你问一个需要计算的问题它靠猜；现在碰到数据问题，它直接算出结果。上传一堆财务报表或实验数据，它真能帮你做统计分析，而不只是写一篇看起来有道理但数字全错的总结。目前对AI Ultra和企业Workspace用户开放。
🎬 Short Video Overviews 60秒竖屏短视频（7月1日上线）
在Studio中选择Video→Short，系统自动从你的来源中提炼一个核心概念，生成60秒TikTok风格的竖屏AI短视频——AI图像+自动旁白。适合把一摞资料快速浓缩成一个能发群里/朋友圈的知识短片。目前仅支持英文。
🗣️ Audio Overview 互动模式
在AI播客播放中随时暂停提问，两位AI主持人会根据你的来源回答，然后接回原本内容。把互动模式当作"流动式口试"——每听到不确定的点就立刻按下按钮请主持人解释或反驳。比被动聆听更能训练理解力。正式开会前跑一遍，每个关键主张都经过你的提问压力测试。

WorkBuddy 实用技巧
⚡ 三要素结构化指令法
让WorkBuddy精准执行的核心公式：做什么 + 有什么 + 怎么样。别写「帮我整理会议纪要」，换成「把D:/会议纪要/0720产品评审.docx整理成一份清单，包含：①每个议题的结论 ②对应责任人和截止日期 ③标记有争议的事项，用表格输出，不需要开场白」。三个要素填满，一轮出成品。
⚡ 多Agent并行，效率翻倍
在同一个对话框里把多个独立任务一次性描述清楚——比如同时要求「整理一份Excel销售数据」+「写一份竞品分析报告」+「做一张活动海报」——WorkBuddy自动拆解成多个Agent并行执行。实测比串行快60%，积分消耗没多多少。
