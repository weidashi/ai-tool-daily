📌 专题周第二天：排版专场。昨天写好了文章，今天给它穿上好看的衣服。md2wechat 是今天的主角——48 个专业主题 + 43 个高级排版模块，一条 CLI 命令把 Markdown 变成可直接复制到微信编辑器的 HTML。不再需要打开 135 编辑器、秀米、新媒体管家手动拖拽调间距。4 个辅助 Skill 覆盖品牌统一、HTML 页面生成、公众号套件排版和社交分发卡片。

公众号排版 Skills
①
md2wechat — Markdown 转微信公众号排版 CLI
geekjourneyx · 48 主题 + 43 排版模块 + AI 配图 + 批量发布 + 多账号管理
⭐ 高星48主题43模块CLI工具
是什么：面向 AI Agent 的微信公众号创作与发布 CLI。写 Markdown，一键生成公众号排版 HTML——48 个专业主题（Minimal、Focus、Elegant、Bold 系列）+ 43 个 :::module 高级排版模块（hero、callout 等）。还内置 write（从想法写文章）、humanize（去 AI 味）、title suggest（标题建议）、generate_cover（AI 配图）等内容生产命令。支持 Codex/Claude/WorkBuddy/Kimi 等多种 Agent。
最适合：用 Markdown 写公众号的人、追求排版一致性矩阵号运营者、想把写作-排版-发布串成一条命令的内容团队。
📦 安装
npm install -g @geekjourneyx/md2wechat && md2wechat config init
💡 初始化后运行 md2wechat doctor 检查配置。Codex 中使用：md2wechat convert article.md --theme elegant-blue --create-draft。个人免费，商业需授权。
🎯 今天就能试：写好的 Markdown 文件 → Codex 里说「用 md2wechat 把 article.md 转成微信 HTML，选 elegant-blue 主题」→ 秒级出排版精美的 HTML，直接复制到微信编辑器。想预览先 md2wechat preview article.md。

📦 md2wechat v2.9.0 发布：新增 brand profile + 多账号支持
md2wechat 最新版支持把长期风格偏好写入 ~/.config/md2wechat/brand.md——以后 Agent 写作和排版时自动读取品牌风格。还新增了命名多公众号账号功能，适合矩阵号运营者。一条命令 md2wechat config wechat-accounts 查看本地账号配置。
⚡ 技巧：md2wechat 的 43 个排版模块怎么用
在 Markdown 里用 :::hero、:::callout 等模块语法插入高级排版组件——比如 :::hero 生成带 eyebrow+title+subtitle 的开篇大图区，:::callout 生成侧边强调框。先用 md2wechat layout list 看全部模块，再用 md2wechat layout show hero 看具体用法。比手动调 CSS 简洁 10 倍。

NotebookLM & Notion 每日进步
🧪 NotebookLM — 一键导出为 PPT / PDF
Studio 模式下的导出功能已经支持 PPT、PDF、Excel 和图片四种格式。做完选题研究后，导出 PDF 发给团队审稿；做完竞品分析后，导出 PPT 直接上台讲。不再需要手动把笔记搬进 PowerPoint。
🧪 NotebookLM — Goals 目标化对话
给笔记本设定固定「目标」和「语气风格」——比如目标设为「帮我写公众号选题报告」，语气「简洁商务风格」。之后这个笔记本里的所有对话 AI 都按这个方向回答。不用每次重新描述需求。
🧪 NotebookLM — Deep Research：让 AI 替你翻 50 篇论文
遇到需要大量资料支撑的选题，开启 Deep Research 模式——NotebookLM 会自动搜索相关论文、报告、新闻，逐条提取关键数据，最后综合成一份带引用的研究报告。对公众号深度文章特别有用。
📋 Notion — External Agents API：让 Codex 直接操作你的工作区
Notion 已开放 External Agents API。外部 AI（Codex、Claude 等）可以直接读写你的 Notion 页面和数据库。对公众号运营者来说，这意味着选题灵感库在 Notion 里、文章草稿也在 Notion 里——Codex 可以自动在这两个地方来回搬运，不用你手动复制粘贴。
