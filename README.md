# AI 工具每日速递

每天 5 分钟，掌握 Codex Skills、NotebookLM、Notion、WorkBuddy 的最新实用技巧。

## 内容板块

- **📘 Codex Skills 每日精选** — 4-5 个高质量公开 Skill，附安装命令和实操步骤
- **🆕 Codex 官方动态** — 新功能、使用提升技巧
- **📗 NotebookLM & Notion 每日进步** — 3 个操作窍门 + 1 个实用功能
- **📙 WorkBuddy 实用技巧** — 快捷指令、自动化示例

## 方向

聚焦日常知识工作 — 写作、PPT、阅读、分发、自动化。不涉及编程/CI/CD/测试。

## 生成方式

由 WorkBuddy 每天早上自动生成，内容来源：GitHub 热门 Skill + IMA 知识库 + 网络搜索。
